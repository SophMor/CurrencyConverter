**Proyecto de Conversión de Divisas en Java**

Este proyecto en Java permite a los usuarios convertir diversas divisas a través de la consola. Utiliza la API de Exchange Rate para obtener las tasas de cambio más recientes, permitiendo así convertir una moneda base a una moneda deseada. El usuario puede ingresar el nombre de la moneda base, la moneda objetivo y la cantidad que desea convertir.

**Requisitos**
JDK 11 o superior.
Conexión a Internet para acceder a la API de Exchange Rate.

El desarrollo de este proyecto fue mediante INTELLIJ IDEA
